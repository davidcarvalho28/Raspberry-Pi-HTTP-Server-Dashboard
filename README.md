# 📊 Raspberry Pi Dashboard

A lightweight HTTP server that displays real-time system stats from your Raspberry Pi using a Flask-based web dashboard.

## 🔧 Features

- 🌡️ **CPU Temperature**
- ⚙️ **CPU Usage**
- 🧠 **RAM Usage**
- 💾 **Disk Usage**
